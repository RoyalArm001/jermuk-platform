Install Jermuk Travel

Այս էջը հասանելի է միայն հավելվածի (PWA) տարբերակով։
Տեղադրիր՝ որպեսզի հետհաշվարկը բոլորի մոտ նույնը լինի և ավտոմատ թարմացվի։

iPhone (Safari)

  1. Բացիր Safari-ում
  2. Սեղմիր Share (քառակուսի ↑)
  3. Ընտրիր Add to Home Screen
  4. Բացի՛ր հավելվածը Home Screen-ից

* iPhone-ում “Install” prompt չկա, պետք է Add to Home Screen։
